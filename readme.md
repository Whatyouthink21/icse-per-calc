# ICSE Percentage Calculator

A dynamic, ICSE-specific percentage calculator with a hacker green theme and Matrix-style animation.

## Features
- English compulsory + best 4 optional subjects
- Science & Social Studies internal averaging
- Matrix-style animated background ("010101 CONGRATS ICSE")
- Green hacker UI, glowing inputs
- ICSE logo support

## Deployment
Static site, deployable via GitHub + Vercel.
